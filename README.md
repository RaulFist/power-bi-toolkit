# 📊 Power BI Toolkit

> Colección de herramientas, templates, fórmulas DAX y automatizaciones con IA para Power BI profesional.

## 🚀 Contenido

### 📁 Estructura
```
power-bi-toolkit/
├── dashboards/          # Templates .pbix listos para usar
├── dax-formulas/        # Biblioteca de fórmulas DAX avanzadas
├── python-scripts/      # Scripts Python para conexión y transformación
├── ai-integration/      # IA generativa integrada en Power BI
└── docs/               # Guías y documentación
```

## 📌 Casos de Uso
- Dashboards ejecutivos con KPIs en tiempo real
- Integración de Python y R para análisis predictivo
- Fórmulas DAX para cálculos complejos (YTD, LY, rolling avg)
- Conexión con APIs externas y bases de datos
- Power BI + Claude/Gemini para análisis conversacional
